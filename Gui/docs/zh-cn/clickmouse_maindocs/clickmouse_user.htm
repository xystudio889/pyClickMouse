介绍

一款快捷，使用python制作的鼠标连点器。目前项目处于早期阶段，功能还不完善，欢迎大家提出宝贵意见。

这个软件可以有较多的版本，基本都是C/C++调用版本、python调用版本和命令行交互版本。

前往releases下载，更新需要替换clickMouse.exe文件。

使用方法

鼠标连点，目前支持左键和右键。
下方的输入框输入间隔，再选择想要点击的类型即可开始连点。

使用优先级

A[exe] --> B[交互式命令行]
鼠标连点器会一直保持运行，直到关闭程序或手动停止。
目前支持暂停和停止功能。
